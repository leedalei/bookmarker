## 一个书签展示器
致力于打造东半球最好的谷歌浏览器收藏夹插件

## 项目构建
```
git clone git@github.com:leedalei/bookmarker.git

cd bookmarker

npm install

npm run build
```
注：每次更新js文件后都需要重新build，因为index.html引用的是webpack打包以后的包。

## 使用方法
设置-扩展程序-开发者模式-加载已解压的应用程序-选择这个文件夹
听到”吧唧“一声后，打开新标签，即可畅享
